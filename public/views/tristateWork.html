Tristate G&T Renewable Energy VR Experience

Project Description

This is a project I worked on with Reality Garage (based in Boulder, Colorado) over the past few months. Our client was Tristate Generation & Transmission, and we were tasked with building a Virtual Reality Guided tour of their renewable energy Co-op program. We built it to integrate with our Inspire XR platform (I have a few pages describing it on this site), which imposed a lot of unique technical constraints on the art we could use.

We built this experience to run on Android phones (with the Samsung Gear VR), so we landed on a low poly art style to limit the complexity of the assets. Turns out VR is pretty computationally expensive.

The phone's processing power wasn't our only limitation, however, as we needed to be able to animate, package, and export all the assets so that we could import, place, and use them at runtime (A big pain, but essential to the InspireXR platform's operation). We ended up working with Unity's Asset Bundles system to accomplish this - but we had to be efficient to load everything in a reasonable time.

This was my largest project thus far working as a Technical Artist - I led all asset creation and animation and worked to make every scene look good in VR, with the constraints of the platform.

Tools Used
